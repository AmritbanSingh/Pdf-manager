📚 PDF Manager Help

🚀 Getting Started

  • Upload PDF files by dragging them to the upload area or clicking to browse
  • Create folders to organize your documents
  • Click on any PDF to view it in the built-in viewer
  • Right-click on files for more options (rename, move, delete)

⌨️ Keyboard Shortcuts

  • Ctrl/Cmd + F - Search files
  • Delete - Delete selected file
  • F2 - Rename selected file
  • Escape - Close viewer or modal

🌐 Deploy to GitHub Pages

  1. Create a new repository on GitHub
  2. Upload these files: index.html, style.css, app.js
  3. Go to Settings > Pages
  4. Select "Deploy from a branch" and choose "main"
  5. Your PDF Manager will be available at: https://yourusername.github.io/repositoryname
Note: This app stores data locally in your browser. Each deployment will have its own separate storage.

💾 Data Management

  • Export Data: Download all your files and folders as a backup
  • Import Data: Restore from a previous backup
  • Storage: Files are stored in your browser's localStorage (max ~10MB total)
